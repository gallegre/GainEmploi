# DT_GainEmploi
La reprise d'emploi est-elle rémunératrice par rapport aux prestations sociales ? 
Document de travail OFCE sur les gains à la reprise d'emploi
Méthode. 
Nous opérons par cas-type, à partir de la maquette Sofi (2024).La maquette est disponible ici : https://github.com/murielpucci/SOFI.
Les résultats utilisés sont dans le fichier "data/Sofi2024". Les hypothèses des cas-types sont commentées dans le texte.
